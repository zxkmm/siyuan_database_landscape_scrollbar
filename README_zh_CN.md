# 块横向滚动条插件

对于长度超过屏幕长度，宽度超过屏幕宽度的块（例如数据库，代码块等），思源只在块底部提供一个横向滚动条，如果你只想用鼠标操作的话，你只能竖向滚动到块最底部，拖动滚动条，然后竖向滚动回原来的位置。  

此插件的作用是，如果插件检测到你的块超长超宽，会在屏幕底部添加一个横向滚动条，你可以通过那个滚动条来进行横向滚动，不需要使用块底部的滚动条。
